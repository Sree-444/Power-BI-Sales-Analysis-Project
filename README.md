# 📊 Sales Analysis Dashboard (Power BI)

## 📝 Project Overview
This project analyzes retail sales data using **Microsoft Power BI** to generate insights on sales performance, profitability, and product trends.

---

## 🚀 Features
- 📌 KPI Cards (Sales, Profit, Quantity)
- 📈 Sales Trend Analysis
- 🌍 Region-wise Performance
- 🏆 Top Products Analysis
- 🗂️ Category Contribution
- 🎛️ Interactive Slicers

---

## 🛠️ Tools Used
- Microsoft Power BI
- Excel Dataset
- DAX (Data Analysis Expressions)

---

## 📂 Files Included
- 📁 Power BI Report File (.pbix)
- 📄 Dataset (Excel/CSV)
- 📘 Project Documentation

---

## 💡 Insights
- Identifies top-performing products  
- Tracks sales trends over time  
- Helps in data-driven decision-making using visualization  

---

## 📌 Conclusion
This dashboard provides a clear and interactive way to analyze sales data, enabling better business decisions and performance tracking.
